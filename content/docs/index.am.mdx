Temetro ሆስፒታሎች ታካሚዎችን፣ ቀጠሮዎችን፣ የሕክምና መዝገቦችን እና ሰራተኞችን ከአንድ በራስ-የሚስተናገድ ዳሽቦርድ እንዲቆጣጠሩ ያግዛል።

## Temetro ለምን?

በብዙ የአፍሪካ ሁኔታዎች የጤና ሶፍትዌር አሁንም ውድ ነው፣ ከክላውድ በጣም የተጠጋ ነው፣ ወይም ከአካባቢ መሠረተ ልማት እውነታ ጋር በቂ አይስማማም። Temetro ግን በተለየ መንገድ ተዘጋጅቷል፦

- **ነጻ እና ክፍት ምንጭ** ስለሆነ ሆስፒታሎች የሚጠቀሙበትን ሶፍትዌር ማየት፣ ማስተካከል እና ባለቤትነት መያዝ ይችላሉ።
- **በራስ-የሚስተናገድ** ስለሆነ የታካሚ መረጃ በሆስፒታሉ ቁጥጥር ስር ባለ መሠረተ ልማት ውስጥ ይቆያል።
- **ለማስጀመር ቀላል** ስለሆነ አነስተኛ የIT ቡድኖች በ Docker Compose ሊያስተዳድሩት ይችላሉ።
- **ለሀብት ውስን አካባቢዎች ተግባራዊ** ስለሆነ ግልጽ በይነገጽና አነስተኛ የስርዓት ጫና አለው።

## ፈጣን ጅምር

```bash
git clone https://github.com/temetro/temetro-deploy
cd temetro-deploy
cp .env.example .env
docker compose up -d
```

ዳሽቦርዱን ለመግባት `http://localhost:3000` እና የሰነዶች ጣቢያውን ለማሰስ `http://localhost:3001/docs` ይክፈቱ።

## Temetro ለማን ይረባል?

Temetro ከዚህ ዓይነት ተጠቃሚዎች ጋር ጥሩ ተስማሚ ነው፦

- የግል ውስጣዊ ዳሽቦርድ የሚፈልጉ ወረዳ እና ክልል ሆስፒታሎች
- ውስን የሶፍትዌር በጀት ያላቸው ሚሽን ሆስፒታሎች እና ለትርፍ ያልተቋቋሙ ተቋማት
- ከምዝገባ ሶፍትዌር ይልቅ ቀላል በራስ-የሚስተናገድ መሠረተ ልማትን የሚመርጡ የIT ቡድኖች
- የሚያስፋፉትን መሠረት የሚፈልጉ ልማት ባለሙያዎችና አስፈጻሚዎች

## የሚያገኙት

የመጀመሪያው አወቃቀር እነዚህን ያካትታል፦

- በ Next.js የተገነባ ምላሽ ሰጪ የሆስፒታል ዳሽቦርድ
- በ Better Auth የተመሰረተ የኢሜይል እና የይለፍ ቃል ማረጋገጫ
- ለታካሚዎች፣ ለቀጠሮዎች እና ለሰራተኞች የስራ ሂደቶች የ PostgreSQL እና Prisma የውሂብ አወቃቀሮች
- ሕዝብ የሚያየው ማስታወቂያ ገጽ
- ሙሉ የሰነዶች ጣቢያ
- ከ NGINX መስመር ማስተላለፍ ጋር የ Docker Compose የማሰማሪያ ፋይሎች
